Kebijakan Privasi

Terakhir diperbarui: 2026-05-09

Ringkasan Singkat

Waktu Sejak adalah aplikasi sepenuhnya offline. Semua data Anda tersimpan secara lokal di perangkat Anda. Kami tidak mengumpulkan, mengirim, atau menyimpan data apa pun di server kami. Tidak ada pelacakan, tidak ada analitik, tidak ada SDK pihak ketiga.

1. Data yang Kami Kumpulkan

Tidak ada. Waktu Sejak tidak mengumpulkan, mengirimkan, atau menyimpan data pengguna apa pun di server kami. Aplikasi ini tidak memiliki akun pengguna, tidak melakukan permintaan jaringan ke layanan kami, dan tidak mengirim telemetri.

2. Penyimpanan Data

Semua event, preferensi, dan pengaturan yang Anda buat di Waktu Sejak disimpan di area penyimpanan privat aplikasi pada perangkat Anda. Data ini hanya dapat diakses oleh aplikasi Waktu Sejak.

Mencopot pemasangan (uninstall) aplikasi akan menghapus seluruh data Anda secara permanen, kecuali file ekspor JSON yang telah Anda simpan sendiri ke lokasi pilihan Anda.

3. Ekspor & Impor Data

Waktu Sejak menyediakan fitur untuk mengekspor data Anda ke file JSON dan mengimpornya kembali. Operasi ini sepenuhnya manual dan dijalankan oleh Anda.

  • File ekspor disimpan ke lokasi yang Anda pilih sendiri di perangkat.
  • File ekspor tidak pernah dikirim ke pengembang atau pihak ketiga mana pun.
  • Anda bertanggung jawab penuh atas keamanan dan distribusi file ekspor Anda.

4. Izin Aplikasi

Waktu Sejak hanya meminta izin yang diperlukan untuk fungsi ekspor dan impor file (jika diminta oleh sistem operasi Anda). Aplikasi ini tidak meminta:

  • Akses internet
  • Notifikasi
  • Lokasi
  • Kontak, kalender, kamera, mikrofon, atau galeri foto
  • Pengidentifikasi perangkat (advertising ID, IMEI, dsb.)

5. Pihak Ketiga

Waktu Sejak tidak menggunakan layanan analitik (mis. Google Analytics, Firebase Analytics), pelaporan crash pihak ketiga (mis. Crashlytics, Sentry), iklan, atau SDK pihak ketiga lainnya yang mengirim data ke luar perangkat.

6. Anak-anak

Aplikasi ini aman digunakan oleh segala usia. Karena kami tidak mengumpulkan data apa pun, kami juga tidak mengumpulkan data dari anak-anak.

7. Perubahan Kebijakan Ini

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Setiap perubahan akan ditampilkan di halaman ini beserta tanggal "Terakhir diperbarui" yang baru. Karena aplikasi tidak terhubung ke internet, perubahan kebijakan tidak memengaruhi data yang sudah ada di perangkat Anda.
